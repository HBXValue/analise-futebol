Scout Football

Cadastro global de futebol

Este MVP organiza países, divisões, clubes e atletas em uma base pronta para crescer. O primeiro foco é o Brasil, com as Séries A, B, C e D, usando o Django Admin como painel operacional simples para cadastro manual.

Passo 1 Cadastre ou revise o país Brasil e suas divisões nacionais.
Passo 2 Adicione os clubes por divisão com filtros e busca no admin.
Passo 3 Prepare o histórico de atletas para a próxima fase do projeto.
